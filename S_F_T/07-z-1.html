7.1.2 Задача 2

Имеются следующие данные импорта нефти условной страны:

Определите относительное изменение средней цены импортной нефти в целом и вследствие факторов.

Оцените структурные сдвиги импорта с помощью коэффициентов:

  1. Гатева \[ \begin{equation} K_G = \sqrt{ \frac {\sum(d_1-d_0)^2} {\sum(d_1^2-d_0^2)} } \end{equation} \]
  2. Салаи \[ \begin{equation} K_S = \sqrt{ \frac{ \sum { (\frac {d_1-d_0} {d_1+d_0})^2 } } {n} } \end{equation} \]
  3. Рябцева \[ \begin{equation} K_R = \sqrt{ \frac {\sum(d_1-d_0)^2} {\sum(d_1+d_0)^2} } \end{equation} \]

Напишите краткие выводы.
